# Agentic Engineering Framework

![Agentic Engineering Framework hero](./assets/agentic-layer-hero.png)

A framework for building the "agentic layer" that turns ad-hoc AI coding sessions into a repeatable engineering process.

[![Docs site](https://img.shields.io/badge/docs-alejtraws.github.io-blue?logo=githubpages)](https://alejtraws.github.io/agentic-engineering-framework/)
[![Last commit](https://img.shields.io/github/last-commit/alejtraws/agentic-engineering-framework)](https://github.com/alejtraws/agentic-engineering-framework/commits)

## 📖 Documentation

**[Read the full guide → https://alejtraws.github.io/agentic-engineering-framework/](https://alejtraws.github.io/agentic-engineering-framework/)**

The site covers:

- **Concepts** — agent patterns, ASE, prompt engineering, self-healing, tool permissions, artifact chaining
- **Framework** — architecture, principles, the 3-layer model (agentic layer / engine / codebase)
- **Workflow** — the 8 phases (Intent → Plan → Build → Test → Review → Document → Deploy → Monitor)
- **Agentic layer** — prompt templates, quality gates, tool integrations, workflow config, commands & skills
- **Adoption** — maturity path, getting started, customization, by team size
- **Reference** — glossary, prerequisites, workflow schema, troubleshooting

## 🗂️ Samples

The framework is documentation-first; working reference implementations live in a separate repo:

**→ [aws-samples/sample-agentic-engineering-framework](https://github.com/aws-samples/sample-agentic-engineering-framework)**

| Sample | Autonomy | Best for |
|---|---|---|
| [`human-in-loop`](https://github.com/aws-samples/sample-agentic-engineering-framework/tree/main/human-in-loop) | L1 Assisted | Learning the AEF patterns by hand |
| [`automated-kiro`](https://github.com/aws-samples/sample-agentic-engineering-framework/tree/main/automated-kiro) | L1–L4 capable | Running AEF autonomously, configurable per team |
| [`sprint-runner`](https://github.com/aws-samples/sample-agentic-engineering-framework/tree/main/sprint-runner) | L3 Autonomous | Installing AEF into existing repos; sprint-batch delivery |
